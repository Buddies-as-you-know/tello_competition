# tello_competition

9月に会津大学で行われるフライングドローンチャレンジ大会のために作成しました。

## 操作
1. 離陸地点にドローンを置く
2. `main.py`の`color_code`を検出したい色に設定(`R`=赤, `B`=青, `G`=緑)
3. マニュアルおよびオートモードコマンドで操作する

|key| 操作 |
|---|----|
| 0 | 手動 |
| 1 | 離陸 |
| 2 | 窓侵入 |
| 3 | マッピング |
| 4 | ライントレース |
| 5 | 着陸 |
| 6 | 自動マッピング |
|---|----|
| g | 移動距離を20->30->100 |
| v | 旋回角度を10->30->90 |
| b | 昇降距離を30->50 |
| x | 960*720 の画像を取得 |
